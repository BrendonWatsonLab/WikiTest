Watson Lab Behavioral Box : phoPythonVideoFileParser

Created by Pho Hale, last modified on Jul 21, 2020

https://github.com/BrendonWatsonLab/phoPythonVideoFileParser

This is basically an old version of the timeline software, and it does similar things without the interface (at the benefit of being a shorter and simpler script) although it may be out of sync with the latest standards.

Analysis software written in Python.

A series of python scripts that load video files created by the Basler Pylon software and parse their timestamps and duration.

Also produces a GUI representation of the Labjack Events (loaded from a single concatenated .txt file) and the video files path.


"onesEventFormat" has a '1' value anywhere an event occurred, and zeros elsewhere. These are computed by checking falling edges to find transitions. Has same dimensionality as inputEventFormat.




1. Modify "labjackFilePath" variable in the "main.py" script to reflect the concatenated .CSV file path. PhoLabjackCSVHelper for more info.

2. Modify "videoFileSearchPath" variable in the "main.py" script to reflect the folder containing the video files.
