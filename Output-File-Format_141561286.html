Watson Lab Behavioral Box : Output File Format

Created by Pho Hale, last modified on Oct 09, 2019

A new .CSV output file is created for each found Labjack when the phoBehavioralBoxLabjackController software is started.

out_file_s{recordingLabjackSerialNumber}_{unix_Epoch_Timestamp}.csv
ComponentDescription
recordingLabjackSerialNumberThe serial number of the Labjack that the file belongs to
unix_Epoch_Timestamp

This timestamp corresponds to the start of the current instance of the phoBehavioralBoxLabjackController software. This value is represented in terms of the number of milliseconds elapsed since the unix epoch (00/00/1970 0:00 GMT).Â 

File Contents

The first line of any .CSV file will be a header describing the comma-separated values.

The second line is an entry containing all '1' values, corresponding to the moment of initialization for a given Labjack.
