Watson Lab Behavioral Box : All 4 dispensers do not work

Created by Anjesh Ghimire, last modified on Dec 04, 2020

Problem

All four dispensers do not work after beam break/finger poke. 

Solution

1) Unplug the Arduino power. 

2) Check to make sure power LED indicator for Arduino is off. (Green = on)

  • If power LED indicator is still on after unplugging power, the Arduino is still receiving power from another source. 
    • If the Arduino is connected to a computer via USB, it will power the Arduino.
  • Unplug the other source of power.

 3) Plug in the Arduino again. All four dispensers should now work.

4)  If dispensers still do not work, unplug power to Arduino again. Also, unplug the power to the motor shield.

(ADD PIC OF THIS):

5) If dispensers still do not work, try another Ardunio:

6) Use some force to separate motor shield (blue board) from Arduino (green board) 

7) Align motor shield (blue) with new Ardunio. Hold the Arduino such that the Arduino power plug is closest to you. Align the motor shield such that the pin to the furthest right on the motor shield will insert into the slot labeled "A5" on the Arduino. (See below)

8) Following the "Arduino-based Diagnostic Procedure," Upload Arduino code (not in diagnostic mode) to new Ardunio. (TO BE FINISHED)

9) Finger poke to see if the dispensers work properly.




9) If dispensers still do not work, try another motor shield.
